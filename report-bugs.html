Report bugs or errors with Payphone Dialler

If you are experiencing any of the following issues, please review the support solutions below:

  • Interface won't load
  • On-screen error message
  • Buttons won't work
  • Buggy or glitches

Contacting support

  1. To begin your report, visit the Payphone Dialler page.
  2. Go to payfonemap.com/phreak and wait for the interface to load.
*NOTE: If the interface will not load, if possible, please provide support with a screenshot of your session.
  1. If you are able to access the interface, go to the top, review the User Session Count (e.g., USC: 12345A), and make a note of it.
  2. Send us an email at support@payfonemap.com with your USC number and/or screenshot in the subject line (e.g., PFD Help - USC: 12345A).
