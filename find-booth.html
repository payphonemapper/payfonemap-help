How to find a booth on Fonectrix

Browsing using the Global Index search

  1. Visit www.fonectrix.info and be sure you are on the INDEX page.
  2. Scroll down to the Global Index section and select your desired region.
*NOTE: You can also browse using the search bar by searching the location or the Booth ID number.

A list of individual payphone booths appears for that location. Each entry includes:

  • Booth ID (e.g., CA-ON-TOR-12345)
  • Type (e.g., Bell Payphone)
  • Observation (street name, city, postal code and/or coordinates)
  • Directions (option to open in Google Maps)
  • Additional notes (if available)

Browsing using the NEAREST search

  1. On the main INDEX page, click on NEAREST in the navigation menu.
  2. Click on Find the Nearest Booths to find payphones within your geographical area.
  3. Once your approximate location is determined, you will be presented with a list of phones by distance.
*NOTE: The range displayed is based on the straight-line distance between your device and the nearest booth.

To get map directions to a specific booth, click on Open in Maps to get full road directions using Google Maps.
