How to play The FONE Game

*NOTE: The FONE Game can only be properly displayed on a computer. You may run into issues playing on a smartphone or tablet.

1. Launch the game

To get started playing, visit The FONE Game page.

Go to payfonemap.com/game and wait for the game to load.

2. Using the on-screen controls

Use the on-screen directional pad or your arrow keys to navigate the maze.

*NOTE: Additionally, you may also choose to use your WASD keys. To use them, be sure to click into the game interface. The game may not function or display properly in a legacy browser. Be sure your browser supports HTML5 (e.g., Chrome, Firefox, Opera).

3. Playing the game

The objective of the game is to pick up all of the payphones in the maze. Be sure you answer the right payphone, or you might get disconnected.

  • When the game session begins, a coordinate will be provided.
  • Pick up the payphone at the coordinate listed to avoid getting blocked.
  • The game will proceed to the next level after all phones are picked up.
  • You will need to reset your session if you become blocked from the payphone.
